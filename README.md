# PingCast Client 

## Começando

```
yarn install
yarn start
```

## Tecnologias utilizadas

* [React.js](https://pt-br.reactjs.org/)
* [React Icons](https://react-icons.github.io/react-icons)
* [Boostrap React](https://react-bootstrap.netlify.app/getting-started/introduction/)
